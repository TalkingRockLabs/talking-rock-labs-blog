# Talking Rock Labs

A collaborative security research blog powered by Jekyll and hosted on GitHub Pages.

## About

Talking Rock Labs is a security research initiative focused on offensive security, vulnerability research, and security tool development. This blog serves as a platform for sharing research findings, technical writeups, and security insights.

## Features

- Modern, responsive design
- Clean, readable layout optimized for technical content
- Syntax highlighting for code blocks
- Category-based post organization
- RSS feed support
- Mobile-friendly navigation

## Local Development

To run this blog locally:

```bash
# Install dependencies
bundle install

# Serve the site locally
bundle exec jekyll serve

# Visit http://localhost:4000
```

## Structure

- `_posts/` - Blog posts in Markdown format
- `_layouts/` - HTML templates for pages and posts
- `_includes/` - Reusable HTML components
- `public/css/` - Stylesheets
- `public/images/` - Images and assets

## Contributing

We welcome contributions from security researchers. To contribute:

1. Fork this repository
2. Create a new branch for your post or changes
3. Add your post to `_posts/` following the naming convention: `YYYY-MM-DD-title.md`
4. Submit a pull request
